Clean .repository before build: This option will cause the plugin to delete all files in the workspace including the .repository (relevant to Maven builds only).
As of perforce-plugin 1.1.1 .repository is skipped
If you wish to remain the default Cleanup behaviour and clean the entire workspace use this option.
This option will also cause the plugin to perform a force sync to retrieve clean files from the perforce server.

Please note:

      This option will only be set if "Clean Workspace Before Build" above is available.
      This option can also be triggered by creating a Boolean parameter on the build called "P4CLEANREPOINWORKSPACE"
      Considering the "P4CLEANWORKSPACE" is also instantiated.
